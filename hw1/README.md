作业内容总结

一、课程要点
1. 各种搜索算法本质相同，主要区别在于边缘（fringe）策略不同。
2. 可以把所有边缘抽象为优先级队列，节点的优先级决定访问顺序。
3. 对 DFS 和 BFS 来说，栈、队列都可以理解为直接实现优先级逻辑的数据结构。
4. 因此可以尝试写一个通用搜索框架，通过传入不同的可变队列对象来实现不同搜索算法。

二、作业要求
1. 使用 Python 实现以下内容：
   - 图（约 10 个节点、若干连接）
   - 树
   - 栈
   - 队列
   - DFS（深度优先搜索）
   - BFS（广度优先搜索）
   - UCS（一致代价搜索）
2. 对实现的搜索算法进行说明，内容至少包括：
   - 完备性
   - 最优性
   - 空间复杂度
   - 时间复杂度

三、可理解为最终交付内容
1. Python 源代码。
2. 用于测试搜索算法的数据结构或示例图/树。
3. 一份对 DFS、BFS、UCS 性质和复杂度的文字说明。
